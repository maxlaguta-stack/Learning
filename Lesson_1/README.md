# Lesson 1 — Условные операторы (if/elif/else)

## Описание
Это моя первая программа на Python в рамках обучения с наставником.  
Программа проверяет возможность доступа в компьютерный клуб в зависимости от:
- возраста
- наличия скидочной карты
- времени посещения

## Что было отработано
- Условные операторы `if`, `elif`, `else`
- Логические операторы (`and`, `or`, `not`)
- Ввод данных через `input()`
- Преобразование типов (`int`, `str`)
- Обработка некорректного ввода
